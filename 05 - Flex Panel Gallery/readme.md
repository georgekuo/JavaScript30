### flex 縮寫屬性 
```
//flex-direction 與 flex-wrap 可合併為 **flex-flow**
預設值 flex-flow: row nowrap;
```

```
//flex-grow 與 flex-shrink 與 flex-basis 三項可合併為 flex
預設值 flex: 1 0 auto;
```
